Get started

    API Endpoint

        https://api.defistation.io/

The Defistation API provides access to read TVL data of defi project listing on defistation.io.

To use this API, you need an user name and API key. Please contact us at david@stamper.network to get your own API key.

get defiNames


# Here is a curl example
curl -X GET \
  https://api.defistation.io/defiNames \
  -H 'Authorization: Basic XxXxXxXxXxpmMjQ5YmQyMC0zMzkzLTExZWItYmVmZC0yMzRiODhkMjNhMWE='

To get definames you need to make a GET call to the following url :
https://api.defistation.io/defiNames

get bnblockedList


# Here is a curl example
curl -X GET \
https://api.defistation.io/bnblockedList/all?days=30 \
  -H 'Authorization: Basic XxXxXxXxXxpmMjQ5YmQyMC0zMzkzLTExZWItYmVmZC0yMzRiODhkMjNhMWE='

The result has pairs of timestamp and BNB amount locked. To get locked BNB data you need to make a GET call to the following url :
https://api.defistation.io/bnblockedList/all?days=30

Errors

The Defistation API uses the following error Messages:

Error Messages Meaning
[Unauthorized] ... Unvalid Authorization key for the api. This error appears if you didn't input proper Authorization key.
